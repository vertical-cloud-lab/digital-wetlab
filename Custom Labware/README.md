# Custom Labware #
Custom Labware refers to any labware related to the Opentrons OT-2 created specifically and uniquely for the Vertical Cloud Lab. This directory is for the accompanying .json files.
## Creating Custom Labware ##
Opentrons has easy to use instructions and software to make custom labware. Access it here: https://labware.opentrons.com/#/create.

In this process, you will export a .json file with all the necessary information for the Opentrons app to utilize and recognize your labware. Upload these .json files to this repository. 

## Using Custom Labware ##
Opentrons has great docmentation on this: https://support.opentrons.com/s/article/Creating-Custom-Labware-Definitions
Custom labware can be uploaded to the Opentrons app through the Labware Tab.
Custom labware can also be utilized directly in protocol creation, either with the Protocol Designer or the Python API.
