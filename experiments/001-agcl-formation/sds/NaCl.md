# SDS Summary: Sodium Chloride (NaCl)

- **CAS No.:** 7647-14-5
- **Molecular Weight:** 58.44 g/mol
- **Form used in experiment:** 1 M aqueous solution

## GHS Classification

**Not classified as hazardous** according to OSHA Hazard Communication
Standard (29 CFR 1910.1200).

No GHS pictograms, signal word, or hazard statements required.

## Physical Properties

| Property | Value |
|----------|-------|
| Appearance | White crystalline solid (solid); colorless liquid (solution) |
| Odor | Odorless |
| pH (5% solution) | 5.0–9.0 at 25°C |
| Melting point | ~801°C (solid) |
| Boiling point | ~1413°C (solid); ~100°C (dilute solution) |
| Vapor pressure | Negligible (non-volatile) |
| Density | ~1.04 g/cm³ (1 M solution) |
| Flammability | Not flammable |

## First Aid

- **Inhalation:** Move to fresh air if dust is inhaled
- **Skin:** Wash with water
- **Eyes:** Rinse with water for several minutes
- **Ingestion:** Rinse mouth; drink water; seek medical advice if large amount consumed

## PPE

- Safety goggles
- Lab coat
- Gloves (optional for solutions)

## Storage

- Keep container tightly closed
- Store in a dry, well-ventilated area

## Waste Disposal

- Non-hazardous at typical lab concentrations
- Can be collected with silver-containing waste if mixed during experiment
- Follow local regulations
